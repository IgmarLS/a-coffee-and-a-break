El sitio web

Sobre el sitio web

Es para una cafetería que ofrece desayunos, brunch, almuerzos, meriendas y cenas casuales. En el sitio web se puede ver la carta de los productos, se puede hacer reservaciones de mesa o pedidos para takeaway o delivery, además hay más información sobre el lugar.

Arquitectura

Mapa del sitio

El sitio web se compone de siete páginas y se estructura con un encabezado y pie de página igual en cada una de las páginas, mientras que en la sección principal el contenido corresponde a la página en cuestión y por ende varía.

Wireframes

Wireframes desktop y mobile

Se comenzó por estructurar la página de inicio a través de los wireframes para desktop y mobile, ya que esta página serviría de base para estructurar y diseña la demás páginas que componen el sitio web.

Diseño

Paleta cromática

Está compuesta por cuatro colores: marrón y azul como colores principales y el blanco y beige como colores de apoyo y contraste. Con estos colores se quiere transmitir calidez, confianza, elegancia, frescura y tranquilidad como parte de un lugar para hacer una pausa.

Tipografía

Se usan dos tipografías de Google Fonts: una script o manuscrita llamada Great Vibes con la que se quiere dar un estilo elegante que denote calidad y tradición para hacer atractivos los títulos y una sans serif condensada llamada Roboto Condensed para transmitir fiabilidad y modernidad de fácil lectura en textos largos.

Sobre el sitio web final

A pesar de ser un sitio web para un lugar ficticio, se logró tener coherencia entre todos los elementos establecidos en la identidad visual. Además, se usan los temas vistos en la cursada fusionados con otras materias para alcanzar mayor impacto y prolijidad en el resultado final.
